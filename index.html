Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

    • How to save changes and puch to github

    1. Make sure your on the main branch git checkout main
    2. pull down the latest main using git pull origin main
    3. Create new branch git checkout -b feature/nameOfbranch
    4. Now make chnages to the file and save them
    5. Stage your changed using git add -A
    6. commit them git commit -m "message"
    7. Make sure that our current branch is up to date before opening a pull request git pull origin main
    8. Push up all of our changes by using git push origin feature/"nameOfbranch"
    9. From here you will go to github and lick the "New pull request"
    10. Add title and discription
    11. CLick Create Pull request
    12. Them merge pull request
